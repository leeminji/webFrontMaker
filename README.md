# webFrontMaker!

<hr/>
## 프로젝트시작
<pre><code>npm update //모듈 다운로드
npm start //서버실행
</pre></code>

> data/menu.json 에 적힌 메뉴명에 따라 메뉴가 생성

<hr />

## 외부접속열기
<pre><code>npm install ngrok -g //안될때 전역으로 설치
ngrok http 3000 //외부포트생성
</pre></code>

> 외부에서 접속할 수 있는 도메인이 생성

<hr />

## HTML 및 페이지 출력
<pre><code>npm run output</code></pre>

> 사이트 전체 다운로드
